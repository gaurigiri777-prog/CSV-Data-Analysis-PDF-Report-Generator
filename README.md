# 📊 CSV Data Analysis & PDF Report Generator

A Python project that reads CSV data, performs analysis, creates visualizations, and generates a professional PDF report using ReportLab.


<img width="1536" height="1024" alt="Data analysis and report generation composite" src="https://github.com/user-attachments/assets/99801a8b-8e5f-4104-b9fa-33b83f97f304" />

---

## 🚀 Features

- 📂 Load and clean CSV data
- 📅 Handle datetime columns
- 📊 Perform data analysis (groupby, aggregations)
- 📈 Generate visualizations using Matplotlib
- 🧾 Create automated PDF reports
- 🖼️ Embed charts inside PDF

---

## 🛠️ Tech Stack

- Python
- Pandas
- Matplotlib
- ReportLab

---

## 📁 Project Structure

```bash
csv-data-analysis-pdf-report/
│
├── data/                 # Input CSV file
├── outputs/              # Generated charts & PDF
├── src/                  # Python script
├── notebook/             # Jupyter notebook
├── README.md
├── requirements.txt
```

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/csv-data-analysis-pdf-report.git

cd csv-data-analysis-pdf-report

pip install -r requirements.txt
```

---

## ▶️ Usage

### Run Python Script

```bash
python src/report_generator.py
```

### OR Run Notebook

```bash
jupyter notebook src/csv_data_analysis.ipynb
```
